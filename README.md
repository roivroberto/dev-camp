# Fylo

## Getting Started

Install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun run dev
```

## Project Structure

```
├── apps/
│   └── web/           # Next.js frontend
├── packages/
│   ├── backend/       # Convex backend
│   ├── config/        # Shared TypeScript config
│   └── env/           # Environment variable validation
```
